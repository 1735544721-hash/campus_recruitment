第 2 章 相关理论知识和技术介绍

2.1 Spring Boot 框架介绍

Spring Boot 是基于 Spring 生态的轻量级后端框架，强调“约定优于配置”，可快速完成 Web 服务搭建。系统后端采用 Spring Boot 3.2，结合分层架构实现控制层、服务层与数据访问层的清晰划分，既降低了模块耦合，也便于功能扩展与维护。

2.2 Vue.js 前端框架

Vue.js 是渐进式前端框架，具备组件化开发与响应式数据绑定能力，适合构建复杂交互的单页应用。其生态完善，可与路由、状态管理与组件库协同使用，提升界面开发效率与交互一致性。

本系统前端采用 Vue 3 与 Element Plus 组件库实现界面构建，通过路由与状态管理完成职位列表、简历管理、投递记录与消息通知等页面的交互联动。组件化开发使职位卡片、表单与弹窗等界面元素可复用，提升前端维护效率。

2.3 MyBatis-Plus 数据访问框架

MyBatis-Plus 在 MyBatis 基础上提供通用 CRUD 与分页能力，减少重复 SQL 编写。系统在数据访问层引入 MyBatis-Plus，通过实体映射与条件构造器实现职位、简历与投递记录的增删改查，并结合分页查询提升列表检索性能。

2.4 MySQL 数据库

MySQL 作为开源关系型数据库，具备事务处理与索引优化能力，适合承载招聘业务的结构化数据。系统使用 MySQL 存储用户、企业、职位、简历、投递与面试数据，通过主键与索引优化提高查询效率，保障数据一致性与可追溯性。

2.5 Java 语言

Java 具有面向对象、跨平台与多线程特性，适合构建稳定的企业级应用。系统后端以 Java 17 作为开发语言，结合注解与分层架构完成业务建模，确保权限控制、业务校验与事务处理的可靠实现。

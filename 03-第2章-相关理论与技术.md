第 2 章 相关理论知识和技术介绍

2.1 Spring Boot 框架

Spring Boot 提供自动配置与约定优于配置的开发模式，能够快速构建面向业务的 Web 服务。本系统后端基于 Spring Boot 3.2 开发，结合依赖注入与分层架构组织业务逻辑，实现控制层、服务层与数据访问层的清晰分工，便于维护与扩展。

2.2 Vue 3 前端框架

Vue 3 提供组件化开发能力与响应式数据绑定机制，适合构建交互性较强的管理界面。系统前端采用 Vue 3 进行页面组织与路由管理，通过组件复用实现职位列表、简历编辑、投递记录与消息通知等功能页面，提升界面一致性与开发效率。

2.3 Spring Security 与权限控制

校园招聘系统涉及学生、企业与管理员多角色访问，需要明确权限边界。Spring Security 提供基于过滤器链的认证与授权能力，支持统一的登录校验与接口访问控制，确保不同角色只能访问对应资源，提升系统安全性。

2.4 MySQL 数据库

MySQL 具备成熟的事务处理与索引机制，适用于招聘业务中高频读写与多表关联查询场景。系统采用 MySQL 保存用户、企业、职位、简历、投递记录与面试安排等核心数据，并通过主键与索引优化查询性能，保障业务流转效率。

2.5 前后端分离与 REST 接口

前后端分离模式将界面呈现与业务逻辑解耦，后端以 REST 风格接口对外提供数据服务，前端通过接口完成数据获取与状态更新。该模式降低了模块耦合度，便于多端扩展，也有利于后期维护与部署。

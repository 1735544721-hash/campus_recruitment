摘  要

高校就业服务对招聘信息发布、岗位匹配、简历管理与投递跟踪提出了更高的数字化要求。传统依赖线下通知与分散表格的方式在信息触达、岗位筛选与流程追溯方面效率较低，难以满足多主体协同的业务需要。本课题围绕校园招聘系统的设计与实现，构建面向学生、企业与管理员三类角色的统一服务平台。系统采用 B/S 架构，后端基于 Spring Boot 3.2 与 Java 17，实现权限控制、数据访问与业务流程编排；前端使用 Vue 3 构建交互界面；数据层采用 MySQL 完成信息持久化。系统覆盖用户注册登录、企业信息维护、职位发布、简历管理、投递记录、面试安排与消息通知等核心功能，并提供后台管理与统计入口。

功能测试围绕登录鉴权、岗位发布、简历投递、面试管理与消息通知等流程展开，结果表明系统能够支撑校园招聘场景下的业务闭环与信息追溯要求，具有良好的可用性与可扩展性。

关键词：校园招聘；Spring Boot；Vue 3；MySQL；简历管理

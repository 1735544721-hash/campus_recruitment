摘  要

目前高校就业服务存在“岗位信息分散、投递流程割裂、反馈不及时”三大痛点，学生难以在同一平台完成职位检索、简历投递与面试跟踪，企业也缺少统一的简历筛选与沟通入口。针对上述问题，本文提出并实现了基于 Spring Boot 的校园招聘系统：前端以 Vue 3 与 Element Plus 构建，提供职位检索、企业信息展示、简历管理、投递记录、面试安排与消息通知等交互；后端基于 Spring Boot 3.2 与 MySQL，结合 MyBatis-Plus 与 JWT 统一封装用户、企业、职位、简历、投递、面试与消息业务，配合事务与索引优化，保障数据一致与流程可追溯。系统支持学生、企业与管理员三类角色的权限控制，完整记录投递状态与面试结果，并提供管理员端的用户、企业与职位监管入口。功能测试覆盖登录、职位发布、简历投递、面试安排与消息通知等流程，结果表明系统运行稳定，能够提升招聘效率与信息透明度。

关键词：校园招聘；Spring Boot；Vue 3；MySQL；简历管理
